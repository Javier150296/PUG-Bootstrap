About

Learn about the project's history, meet the maintaining teams, and find out how to use JADE-Bootstrap.

History

Originally created by a designer and developer, we hope JADE-Bootstrap will become one of the most popular replacement in the world for writing plain-old HTML with its boring and awfully verbose syntax with the cleaner and faster way to write presentation code.

JADE-Bootstrap was created in mid-2015 by @Rajasegar to overcome his repetitive and boring way of creating the same user interface components for web applications using Bootstrap by writing the same markup again and again. Since then he decided to create a framework based on JADE mixins with a high level of configurability to create beautiful Bootstrap components with very less code in very less time.

This project is still in its beta stage and has to overcome a lot of roadblocks to get to a high level of production usage and availability. But still you can use the project's code to create your own web applications quickly and efficiently if you are comfortable writing JADE.

Team

JADE-Bootstrap is maintained only by the founding team at present and we welcome a group of invaluable core contributors, with the massive support and involvement.
